# Instruction to compile Project

## First of all you have to run 

```
npm i
```

And to make sure that project is working run:
```
npm run dev
```
the entry point is: 
```
src/views/App.tsx
```


To validate that the form is working, you have to open the brower console:

In windows/Ubuntu
```
ctr + alt + i
```
in Mac
```
command + alt + i
```
